## 🚀 System Overview
This project maps my entire digital orbit—from escaping the LinkedIn "ghost" phase to scaling a writing platform to 800,000+ reads. It is built from scratch to reflect a high-tech, deep-space aesthetic, mirroring my focus on continuous growth, AI/ML, and cloud computing.

### 🛸 Core Features
* **Aerospace Particle Canvas:** A custom-engineered, interactive background using HTML5 Canvas and Vanilla JavaScript to simulate data nodes in deep space.
* **Holographic Profile Interface:** A CSS-driven glitch and scanline effect that projects the user portrait as a technological hologram.
* **Dynamic Support Orbit:** Instead of a traditional list, acknowledgments are mapped as an animated planetary orbit, highlighting the foundational support network that fuels my growth.
* **Custom UI Elements:** Glowing cyan borders, glassmorphism panels, and hue-rotated graphical assets to maintain a strict deep-blue aerospace theme.

## 🛠️ Tech Stack
